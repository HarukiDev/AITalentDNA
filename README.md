# AITalentDNA
AITalent DNA is a Website

## Installation
**Git Clone**
```
git clone https://github.com/HarukiDev/AITalentDNA.git
```
**Go to the Directory**
```
cd frontend
```
**Install NPM**
```
npm i
```
**How to run**
```
npm run dev
```
